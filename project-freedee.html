What It Does

PhreeDee is a desktop app (Electron + Python backend) that converts images into 3D meshes using Hunyuan3D models. Everything runs locally on your GPU — no cloud processing, no API keys, no monthly fees. It includes a visual workflow editor (drag-and-drop node graphs), mesh processing tools (smooth, decimate, repair, remesh), an extension system for installing new AI models from GitHub, and a CLI so AI agents can automate 3D generation.

"But I don't have a beefy GPU." — If you have 4GB of VRAM, you're good for the mini models. If you have 12GB, you can run the full suite. If you have less than 4GB... well, there's always the upgrade path. Or PowerPoint. PowerPoint does 3D too, technically.

Features

🧠 AI-Powered Generation

Convert any image into a 3D mesh using Hunyuan3D models. Multiple variants: Mini (6GB), DIT v2.0 (10GB), FLUX.1-schnell (12GB). Pick your VRAM poison.

🎨 Visual Workflow Editor

Drag-and-drop node editor for complex 3D generation pipelines. Connect inputs, outputs, and processing nodes. It's like building with Lego, except the bricks are math.

⚙️ Mesh Processing

Smooth, decimate, repair, and remesh built-in. No need to open Blender just to fix a hole in your mesh. Though Blender would appreciate the company.

🔌 Extension System

Install new AI models from GitHub repos. Each extension is a manifest.json plus runtime files. Click "Install from GitHub," paste the URL, done. It's like npm but for 3D.

💻 CLI & MCP

Automate from terminal or AI agents (Claude Desktop, Codex, OpenCode). Run `generate --image ./input.png --output ./export.glb` and walk away. The agent does the heavy lifting.

🖥️ Desktop App

Electron app for Windows, Linux, and Apple Silicon macOS. Live RAM indicator in the top bar. Workflow validation before run. It's a proper app, not a Jupyter notebook.
