Concept

X-Ray uses four anatomically inspired image layers — background, skeleton, muscle fibres, and skin — stacked from bottom to top. Each layer (except the background) has a toggle switch. When a layer is toggled on, it's visible normally but fades out on hover to reveal what's beneath. When toggled off, it stays hidden regardless of hover.

This creates a progressive dissection effect: enable layers above the one you want to see, then hover to peel them back.

How It Works

Four <img> elements are absolutely positioned and stacked by z-index. Each layer tracks two states: enabled (toggle on/off) and hovering (mouse over the stage).

An enabled layer is visible by default and fades to opacity:0 when the cursor enters the stage. A disabled layer stays at opacity:0. The background layer is always on — it's the base you reveal down to.

Opacity transitions use CSS transition:opacity .15s ease-out for smooth fades. No canvas, no masking, no JavaScript compositing — just stacked images and CSS opacity.
