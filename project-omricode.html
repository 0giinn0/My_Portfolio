What It Does

OmriCode is a standalone Electron desktop app — a fully local AI coding environment with a built-in code editor, terminal, 3D viewer, and an AI chat panel that connects to any LLM provider. Local models via Ollama, cloud APIs, free tiers — you pick. The agent has a full tool registry: read/write/edit files, run bash, search the web, grep codebases. Every edit is undoable. 12 personality profiles change how the agent talks — from Deadpool to GLaDOS to Gordon Ramsay.

"Is it production ready?" — Define production. If you mean "can I use this to build things and break things and cause chaos?" — then yes. If you mean "ISO certified with enterprise SLAs?" — then no, and frankly, ew.

Features

◈ Code Editor

Multi-tab editor with syntax highlighting, split views, and a view pane for 3D models or web preview. Drag dividers to rearrange panels. It's an IDE that went to art school.

◈ AI Chat Panel

Talk to any LLM — local or cloud. Every tool call shows as an expandable card with arguments and a Revert button. Thinking indicators show reasoning cycles left. Slash commands for power users.

◈ Built-in Terminal

Full xterm.js terminal under the editor. PowerShell on Windows, bash/zsh on Linux/macOS. Run commands without leaving the app. Drag the divider to give it room.

◈ 3D Viewer

Three.js-powered viewer loads GLTF/GLB/OBJ/STL files. Drag-and-drop models, orbit camera, auto-fit. All local — no CDN needed. Because sometimes you need to see your code in three dimensions.

◈ Personality Profiles

12 built-in personalities: Architect, Deadpool, Rick Sanchez, Socrates, Tyler Durden, Yoda, GLaDOS, Dr. House, Gordon Ramsay, HAL 9000, Morpheus, Sherlock. Pick one in Settings.

◈ Editor Plugins

VS Code extension, Godot 4 plugin, and Blender add-on. The same AI brain across your entire workflow. TypeScript, GDScript, and Python — one agent, three editors, zero fucks given.

Provider Support

◈ Free Providers

Ollama — fully local, free (your GPU). OpenRouter — free tiers. Google Gemini — free 60 req/min. GitHub Models — free with GitHub account. Groq — free 30 req/min, insane speed.

◈ Paid Providers

OpenAI — when you have money. Anthropic — long context, good vibes. Any OpenAI-compatible endpoint works — LM Studio, LocalAI, your toaster running Linux.
