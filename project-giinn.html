What It Does

Giinn is a Python CLI that turns your terminal into an AI-powered development workstation. Edit files with any LLM, run multi-agent builds where specialized AI roles (Planner, Coder, Reviewer, Tester, Documenter) collaborate on your project, fine-tune models from your coding patterns, build mobile APKs, and deploy — all from one tool.

"But why not just use Copilot?" — Because Copilot doesn't have a GUI with a 3D model viewer and a phone preview pane. Checkmate.

Features

🤖 AI Code Editor

Edit files with OpenAI, Anthropic, Gemini, Ollama, or OpenRouter. AST-aware context extraction — sends only the relevant function, not the whole file. Cost estimation before every edit. Because surprise bills are for phone companies.

🏗️ Multi-Agent Builds

Describe your project and watch 5 specialized agents collaborate: Planner, Coder, Reviewer, Tester, Documenter. Each phase pauses for your approval. It's like managing a team, except they don't ask for raises.

🧠 Built-in RAG

Index your project with embeddings for cross-file context. Auto-updates on file changes. Your edits stay consistent across the entire codebase. The AI remembers so you don't have to.

📱 Mobile APK Builder

Auto-detects Flutter, Expo, React Native, Kivy, Flet, or native Android. Picks the right cloud build provider (EAS, GitHub Actions, Codemagic), triggers the build, polls until done, downloads the artifact, and generates a QR code. It does everything except actually use the app for you.

🎛️ Desktop GUI

PyWebView or browser-based. Spiral hub navigation, built-in code editor, live web preview, Three.js 3D viewer, phone preview with APK controls, AI chat console, multi-agent build output. It's an IDE that went to art school.

🔧 Fine-Tuning

Every edit is logged. Build a training dataset from your actual coding patterns. QLoRA on your local GPU or cloud RunPod. Export to Ollama. Teach the AI to code like you — for better or worse.

Quick Start
# Install pip install giinn # Setup giinn init # Edit a file with AI giinn edit main.py -p "add error handling" # Build a whole project with multi-agent giinn build "REST API for a todo app with FastAPI + SQLite" # Deploy a preview giinn deploy --preview # Open the GUI giinn gui
